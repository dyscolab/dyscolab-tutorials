# Dyscolab tutorials
Tutorials for [dyscolab](https://dyscolab.github.io/) libraries. Uses [pixi](https://pixi.prefix.dev/latest/) to install ensure a reproducible environment. Once pixi is installed in your system run:

```sh
pixi install
```
to install pixi environment. Tp run 
```sh
pixi run open
```
to open [marimo](https://marimo.io/) launch page in your browser or 

```sh
pixi run open [path_to_notebook]
```
to open a specific notebook. They can also be opened in other editors ([see marimo's support](https://docs.marimo.io/guides/editor_features/watching/)), just make sure to choose the "default" environment.
